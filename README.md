## Westerville Archery — Personal Site

A fast, responsive one‑page site for an archery instructor in Westerville, Ohio.

### Quick start
1. Open `index.html` in your browser.
2. Replace placeholder text (name, email, pricing) with your own.
3. Add your images to `assets/`:
   - `hero-archery.jpg` (hero section, ~1600×1200)
   - `coach.jpg` (about section, ~1200×900)
   - `og-image.jpg` (social share image, 1200×630)
   - `favicon.png` (32×32 or 64×64)

### Customize
- Branding: Update logo in `assets/logo.svg` or swap the file.
- Colors/typography: Edit CSS variables in `styles.css` under `:root`.
- Contact email: In `index.html`, change both `mailto:you@example.com` values.
- Social links: Update `sameAs` array and `<meta property="og:*">` in the `<head>`.
- Business details: Update the JSON‑LD (`@type: SportsActivityLocation`) in `<head>`.

### Deploy
- GitHub Pages: Commit, push, and set the repo to deploy from the root. Your site will be at `https://<username>.github.io/<repo>/`.
- Netlify/Vercel: Drag‑and‑drop the folder or link the repo. No build step required.

### Notes
- The contact form uses `mailto:` for simplicity. If you prefer a hosted form backend, services like Formspree or Basin work well; replace the `action` on the form.
- Images are optional for first preview; add them later for best results.
